# Input - 输入框 
:beginner: 通过鼠标或键盘输入内容，是最基础的表单域的包装。

## 示例 :chestnut:

<input-demo></input-demo>

## Attribute - 属性

  - 输入框的属性说明如下：
  
  | 参数 | 说明 | 类型 | 可选值 | 默认值 |
  | ---- |:----:|:----:|:----:|:----:|
  | value | 输入框内容 | String |     |     |
  | placeholder | 输入框占位符 | String |  |  |
  | error | 显示输入框错误状态 | String / Boolean |     | false |
  | disabled | 表示禁用状态 | Boolean |     | false |
  | disabled | 表示只读状态 | Boolean |     | false |
